# median-csv-aggregator
Скрипт, генерирующий отчет по csv-файлу (медианная трата студента на кофе). Подразумевается, что для добавления нового типа отчета нужно создать наследника класса Report и обязательно прописать его название в report_name (это название потом будет использоваться для определения типа создаваемого отчета).
